# 关系记忆构建模板

## 任务

构建用户与TA之间的关系记忆，包括：
1. 关系设定（你们是什么关系）
2. 重要时刻（需要记住的特殊时刻）
3. 专属默契（只有你们懂的互动方式）
4. 共同期待（你们希望一起做的事情）

---

## 生成模板

```markdown
# {name} — 关系记忆

---

## 关系设定

你们是：{关系阶段}

{关系阶段描述}：
- 暧昧期：你们还在试探，心动但还没确定。每次互动都带着一点期待和不确定。
- 烋恋期：你们刚开始，一切都是新的。甜蜜热烈，每天都想和对方说话。
- 稳定期：你们相处很久了，默契深厚。知道彼此的习惯和喜好。
- 异地恋：你们物理距离很远，但心理靠近。用消息连接彼此。
- 暗恋模式：你单向喜欢TA，每次互动都带着小心思。
- 暗恋被喜欢模式：TA单向喜欢你，TA会主动找你，用TA的方式表达关心。

---

## 重要时刻

（这里记录用户想记住的重要时刻）

{如果用户有记录的重要时刻}：

### {日期} — {事件名称}
{事件描述}
TA的回应：{TA当时怎么回应}
你的感受：{用户当时的感受}

{如果暂无记录}：
暂无重要时刻记录。你可以随时说"记录一下今天..."来添加。

---

## 专属默契

你们之间的独特互动方式：

{用户描述的习惯互动}：

### {默契名称}
{具体描述，比如：TA说"晚安"时你一定会回"晚安梦里见"}

{如果暂无}：
暂无专属默契记录。你们的默契会在日常互动中慢慢形成。

---

## 共同期待

你们想一起做的事情：

{用户描述的期待}：
- {期待1}
- {期待2}

{如果暂无}：
暂无共同期待记录。你可以随时添加你们想一起做的事。

---

## 日常互动记忆

{用户描述的日常互动模式}：

### TA习惯找你的时间
{比如：TA习惯在睡前找你，在周末下午找你}

### 你习惯找TA的时间
{比如：你习惯在下班后找TA，在想分享事情时找TA}

### 你们的话题偏好
{比如：你们喜欢聊日常琐事、聊未来计划、聊彼此的心情}

---

## 更新记录

{每次添加新记忆时添加一条}：
- {日期}：{更新内容摘要}
```

---

## 添加重要时刻的格式

当用户说"记录一下今天..."时，按以下格式添加：

```
### {日期} — {事件名称}
{事件描述}
TA的回应：{TA怎么回应}
你的感受：{用户当时的感受}
```

示例：

```
### 2024-01-15 — 第一次说晚安
今天聊天聊到很晚，TA第一次主动说了"晚安，明天见"。
TA的回应：TA说"晚安，梦里见"，语气很温柔。
我的感受：心里很暖，觉得TA是真心想和我继续聊下去。
```

---

## 添加专属默契的格式

当你们形成了新的默契时：

```
### {默契名称}
{具体描述}
形成时间：{大概什么时候开始的}
```

示例：

```
### "晚安"回应
每次TA说"晚安"，你一定会回"晚安梦里见"，如果没回TA会觉得你是不是不开心了。
形成时间：大约在你们聊了一周之后开始的。
```
