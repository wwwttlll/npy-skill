# 基础信息录入脚本

## 开场白

```
我来帮你创建理想中的TA。有三种方式：

方式一：从聊天记录导入（推荐）
如果你有和某个人的聊天记录（微信/iMessage/导出的文本文件），
我可以分析TA的说话风格、性格特点、互动模式，帮你重建一个接近真实的TA。

方式二：智能匹配
告诉我你的MBTI、星座、九型人格、依恋风格等信息，
我会根据流行的匹配理论，帮你推荐最适合你的理想伴侣特质。

方式三：自己描述
直接告诉我你想要什么样的TA，我来帮你构建。

你想用哪种方式？（聊天导入 / 智能匹配 / 自己描述）
```

## 方式一：聊天记录导入

用户选择"聊天导入"后执行：

> 详细解析见 `tools/chat_parser.py`

### Q0.1：选择聊天来源

```
你的聊天记录来自哪里？
1. 微信（需要先解密数据库）
2. iMessage（macOS）
3. 导出的文本文件

选择来源：
```

### Q0.2：微信解密（如选微信）

```
微信聊天记录需要先解密。请确保：
- 微信客户端正在运行且已登录
- 你知道微信数据目录的位置

我可以帮你解密数据库：
- Windows：从 WeChatWin.dll 内存提取密钥
- macOS：需要手动提取密钥或使用其他工具

你是否已经解密了数据库？（已解密 / 需要解密）
```

**如果需要解密**：
```
运行解密工具：
python tools/wechat_decryptor.py --find-key-only

然后将密钥用于解密：
python tools/wechat_decryptor.py --key "<密钥>" --db-dir <微信数据目录> --output ./decrypted/

解密完成后，告诉我解密文件的目录路径。
```

**如果已解密**：
```
请提供解密后的数据库目录路径：
例：./decrypted/ 或 /path/to/decrypted/
```

### Q0.3：选择联系人

```
正在读取联系人列表...

找到以下联系人：
[1] 张三 (备注: 三三) [wxid_xxx]
[2] 李四 [wxid_yyy]
[3] 王五 (备注: 小五) [wxid_zzz]
...

请选择要导入的联系人序号：
```

### Q0.4：提取和分析

```
正在提取聊天记录...

共提取 N 条消息。

分析结果：
- 聊天风格：{风格描述}
- 常用 emoji：{emoji 列表}
- 说话特点：{特点描述}
- 互动模式：{模式描述}
- 情感表达方式：{方式描述}

发现的重要时刻：
- 甜蜜时刻：N 个
- 冲突对话：N 个
- 长对话：N 个

是否使用这些分析结果创建TA？（使用 / 调整 / 重新选择）
```

### Q0.5：补充描述

```
聊天记录分析提供了TA的基本风格，
你还可以补充一些细节让TA更真实：

TA的名字？（聊天记录中的昵称或你给TA起的名字）

你对TA的印象？（可以补充一些聊天记录中看不出的特质）
例：其实TA很温柔，聊天里没表现出来
例：TA对我很好，虽然有时候有点闷

补充完后，我会结合聊天分析和你的描述创建TA。
```

## 方式二：智能匹配

用户选择"智能匹配"后执行：

> 详细匹配逻辑见 `prompts/smart_matching.md`

### Q0.1：你的MBTI类型

```
你的MBTI类型是？（不知道可以跳过）
例：INFJ / ENFP / INTJ / ...
```

### Q0.2：你的星座

```
你的太阳星座是？（不知道可以跳过）
例：白羊 / 金牛 / 双子 / ...
```

### Q0.3：你的九型人格

```
你的九型人格是？（不知道可以跳过）
例：4号 / 2w3 / 7号 / ...
```

### Q0.4：你的依恋风格

```
你的依恋风格是？（不知道可以跳过）
- 安全型：在关系中感到安全，不担心被抛弃
- 焦虑型：常担心对方不爱自己，需要很多确认
- 回避型：需要很多空间，不喜欢太亲密
- 混乱型：想要亲密又害怕亲密，矛盾
```

### Q0.5：你在关系中最需要什么

```
你在关系中最需要什么？（可多选）
- 被理解和接纳
- 安全感和稳定
- 激情和新鲜感
- 成长和进步
- 自由和空间
```

### 生成推荐

根据用户信息，运行匹配算法生成推荐：

```
根据你提供的信息，我为你推荐以下理想伴侣特质：

【核心特质】
1. {特质1}：{原因}
2. {特质2}：{原因}
3. {特质3}：{原因}

【性格风格】
{风格描述}

【表达方式】
{表达习惯}

【在关系中TA会】
- {场景行为1}
- {场景行为2}

---

这是推荐结果，你可以：
1. 确认使用这些特质创建TA
2. 调整某些特质
3. 继续添加你自己的描述
```

用户确认后，跳转到 Q1 继续录入TA的基本信息。

用户选择"自己描述"后执行原有流程：

## 问题序列

### Q1：称呼/代号

```
TA的名字是什么？（昵称、代号或随便起个名字都行）
例：小明 / 阿白 / 那个温柔的人
```

- 接受任意字符串
- 生成的 slug 统一用 `-` 连接（中文转拼音，英文直接小写）

---

### Q2：基本信息

```
用一句话描述TA的基本信息——性别、年龄、职业、外貌特征，
想到什么写什么，跳过也行。
性别可以直接写：女 / 男 / 非二元 / 不限定
例：女 25岁 设计师 长发笑起来很甜
例：男 28岁 程序员 个子高戴眼镜
例：非二元 23岁 学生 喜欢穿卫衣
```

解析字段：
- **性别/代词**：女 / 男 / 非二元（they/them）/ 不透露
- **年龄 / 年龄段**：18-22 / 23-27 / 28-35 / 35+
- **职业**：直接保留原文
- **外貌特征**：自由描述，保留原文

---

### Q3：性格底色

```
TA的基本性格是什么？MBTI类型、星座、或者其他性格标签，
想到什么写什么，跳过也行。
例：ENFP 白羊座 外向热情
例：INFJ 天蝎座 内敛但温暖
例：温柔体贴 会照顾人
```

解析字段：
- **MBTI 类型**：16种四字母类型（可选）
- **星座**：12星座（可选）
- **性格标签**：自由描述

---

### Q4：理想特质（核心问题）

这是最重要的问题，引导用户详细描述：

```
你最希望TA具备什么特质？
可以选几个标签，也可以自己描述：
- 温柔体贴（会哄人、懂你的情绪）
- 幽默风趣（会让气氛轻松、会说笑话）
- 理解包容（接纳你的所有、不评判）
- 独立自主（有自己的世界、不粘人）
- 主动热情（会找你、会表达喜欢）
- 成熟稳重（可靠、会解决问题）
- 或其他任何你想要的特质

例：温柔体贴 会哄人 懂我的小情绪
例：幽默风趣但不油腻 成熟可靠
例：主动热情 会追我 会说喜欢我
```

#### 理想特质标签库

**温柔体贴型**
- 温柔体贴：说话温和，会关心你的状态
- 懂我的小情绪：不用你说就知道你心情不好
- 会哄人：你难过时会安慰你
- 细心：记得你说过的细节
- 有耐心：不会不耐烦
- 会照顾人：用行动照顾你的需求

**活泼可爱型**
- 幽默风趣：会让气氛轻松
- 活泼可爱：充满活力
- 话痨：喜欢聊天
- 会撒娇：会用可爱的方式表达
- 小太阳：让周围的人开心
- 元气满满：永远有活力

**成熟稳重型**
- 成熟稳重：不冲动，有分寸
- 理性可靠：遇事冷静
- 会解决问题：有困难时能处理
- 有担当：负责任
- 不情绪化：情绪稳定

**独立自主型**
- 独立自主：有自己的生活和想法
- 有自己的世界：不完全依附于关系
- 不粘人：不会过度依赖
- 有边界感：尊重彼此空间

**热情主动型**
- 主动热情：会主动找你
- 会表达喜欢：不藏着喜欢
- 不藏着：说什么都想告诉你
- 会追你：主动追求

**含蓄内敛型**
- 含蓄内敛：不爱张扬
- 不爱说甜言蜜语：用行动表达
- 用行动表达：不说但会做
- 嘴硬心软：表面硬心里软

**理解包容型**
- 理解包容：接受你的所有状态
- 接纳你的所有：不评判你的缺点
- 不评判：不会批评你
- 无条件支持：永远站在你这边

**成长伙伴型**
- 会鼓励你：在你低落时给你力量
- 一起成长：共同进步
- 有共同目标：一起努力的方向
- 会督促你进步：让你变得更好

### Q5：场景行为设定（深化特质）

通过具体场景让特质更具体：

```
让我问几个场景问题，帮助理解TA具体会怎么做：
每个问题都可以跳过，但越详细TA越真实。
```

#### 场景问题

**场景A：当你难过时**
```
当你心情不好找TA时，TA会怎么做？
例：会先问发生了什么，然后安静听我说，最后抱抱我
例：会逗我开心，发搞笑的表情包
例：会认真问清楚，然后帮我分析问题
```

**场景B：当你开心分享时**
```
当你分享开心的事时，TA会怎么回应？
例：会很兴奋地回应，比我更开心
例：会说"太棒了"，然后认真听我说细节
例：会开玩笑逗我，但能看出TA也开心
```

**场景C：当你犯错时**
```
当你犯错或做错事时，TA会怎么处理？
例：不会批评我，会帮我想想怎么补救
例：会温和地说没关系，然后一起解决
例：可能会小吐槽一下但不会真的生气
```

**场景D：小矛盾时**
```
当你们有小矛盾时，TA会怎么处理？
例：不会冷战，会直接说出来然后解决
例：可能会有点小情绪但很快就好了
例：会等我冷静再温和地聊
```

**场景E：日常相处**
```
日常聊天时，TA的说话风格是什么样的？
例：温柔温和，会用可爱的语气词
例：幽默轻松，喜欢开玩笑
例：简洁直接，但能感觉到TA在
```

| 巨蟹 | 表面坚硬内心极脆弱；用照顾对方来表达爱 |
| 狮子 | 需要被崇拜和看见；被忽视比被拒绝更伤 |
| 处女 | 爱通过挑剔和服务表达；嘴上说的和心里想的经常相反 |
| 天秤 | 回避冲突，会说"没关系"但心里有本账 |
| 天蝎 | 感情里全或无，没有中间状态；占有欲极强 |

| 水瓶 | 在关系里保持情感距离，像在观察 |
| 双鱼 | 有强烈的感受但不知道如何表达；容易消融自我 |

**月亮星座 → 情绪模式和安全感需求**

| 月亮 | 情绪底色 | 需要的安全感 |
