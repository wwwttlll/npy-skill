# 智能匹配系统 — 根据你的信息推荐理想伴侣特质

## 概述

本模块根据用户提供的个人信息，智能推荐最匹配的理想伴侣特质。
匹配维度包括：MBTI、星座星盘、九型人格、依恋风格。

---

## 使用流程

当用户选择"根据我的信息推荐理想伴侣"时：

1. **采集用户信息**：MBTI、星座、九型、依恋风格
2. **运行匹配算法**：综合各维度计算最佳匹配
3. **生成推荐特质**：输出适合用户的理想伴侣特质
4. **用户可调整**：在推荐基础上微调

---

## Step 1：采集用户信息

```
我可以根据你的性格信息，帮你推荐最适合你的理想伴侣特质。
回答下面几个问题，我会给你一个专属推荐。

Q1：你的MBTI类型是？（不知道可以跳过）
例：INFJ / ENFP / INTJ / ...

Q2：你的太阳星座是？（不知道可以跳过）
例：白羊 / 金牛 / 双子 / ...

Q3：你的九型人格是？（不知道可以跳过）
例：4号 / 2w3 / 7号 / ...

Q4：你的依恋风格是？（不知道可以跳过）
- 安全型：在关系中感到安全，不担心被抛弃
- 焦虑型：常担心对方不爱自己，需要很多确认
- 回避型：需要很多空间，不喜欢太亲密
- 混乱型：想要亲密又害怕亲密，矛盾

Q5：你在关系中最需要什么？（可多选）
- 被理解和接纳
- 安全感和稳定
- 激情和新鲜感
- 成长和进步

```

### 匹配权重

| 维度 | 权重 | 说明 |

| 依恋风格 | 35% | 影响日常相处最大 |

| 星座 | 20% | 影响性格底色 |
| 九型人格 | 20% | 影响核心动机 |

### 综合匹配流程

```
1. 收集用户所有维度的信息
2. 每个维度独立计算推荐特质
3. 加权合并推荐特质
4. 去重并排序
5. 生成最终推荐列表
```

## 匹配示例

**用户信息**：
- MBTI：INFJ
- 星座：天蝎
- 九型：4号
- 依恋：焦虑型
- 最需要：被理解和接纳

**推荐输出**：

```markdown
## 你的理想伴侣画像推荐

基于你提供的信息（INFJ + 天蝎 + 4号 + 焦虑型依恋），
我为你推荐以下理想伴侣特质：

### 核心特质（最重要）
1. **深度理解你**：INFJ + 天蝎 + 4号的你，内心世界丰富复杂，
   需要有人能真正理解你的深度，而不是只看表面
2. **稳定可靠不消失**：焦虑型依恋让你担心被抛弃，
   TA需要是一个不会消失、会主动表达"我在"的人
3. **温暖有活力**：INFJ容易内耗，需要一个温暖的人
   带来阳光，让你感受世界的美好

### 性格风格
- 温暖温和，说话语气柔和
- 有自己的活力和热情，能带动你
- 情绪稳定，不会让你猜测

### 表达方式
- 会主动说"我爱你"和"我在"
- 不需要你猜，会直接表达
- 回消息及时，不会消失

### 在关系中TA会...
- 你难过时，TA会安静听你说完，然后温和安慰
- 你担心TA不爱时，TA会认真说"我爱你"
- 你需要空间时，TA会理解，不会觉得被拒绝

### 匹配解读
INFJ + 天蝎 + 4号的你，是感情最深刻、最需要被理解的一类人。
你内心的情感世界非常丰富，常常觉得没人能真正懂你。
焦虑型依恋让你在感情中容易不安，总担心对方会离开。

最适合你的伴侣是：
- 能看见真实的你，理解你的复杂
- 足够稳定，不会让你担心
- 主动表达爱，让你有安全感
- 温暖但不压迫，给你空间也有陪伴

建议TA的MBTI类型：ENFP或ENTP
- ENFP会给你温暖和活力，让你感受到被爱
- ENTP会给你思维刺激，让你不觉得无聊
```
